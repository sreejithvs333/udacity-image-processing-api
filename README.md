# udacity-image-processing-api

A simple Nodejs image processing api project which uses typescript & express

## Scripts

1. Install dependencies

```
npm i
```

2. To build

```
npm run build
```

3. To run development

```
npm run start
```

4. To run production

```
npm run prod
```

5. To test

```
npm run test
```

6. To code format

```
npm run prettier
```

7. To check format

```
npm run prettier-check
```

8. For linting

```
npm run eslint
```
